VLSI CAD Group Index of Useful Tools

This page provides access and links to tools which the Case Western Reserve University, Computer Engineering Department's VLSI CAD Group has developed or found useful.

VHDL Tools

  • VHDL Public Domain Packages : A collection of VHDL packages which duplicate the #include stdio.h, strings.h, stdlib.h, ctype.h libraries. Also, notes on how to write to Unix pipes to interface interactively with external EDA tools (i.e. MATLAB). Download the complete VHDL source C/Unix packages for VHDL Synopsys and Mentor graphic simulators .
  • VHDL-NICE93 : a VHDL-93 beautifier. This is source with LINUX & SunOS binaries.
  • VHDL-NICE : a VHDL-87 beautifier. This is source with LINUX binaries.
  • VHDL-2-C : a VHDL-87 VHDL to C conversion program. This tar file includes LINUX, SunOS, and WIN32 binaries and manual pages.
  • VHDLPROF : a VHDL-87 profiling program. This tar file includes LINUX, SunOS, and WIN32 binaries and manual pages.
  • VHDL2HTML : Converts and colorizes VHDL into HTML written by Jon Connell(jco@egnetz.uebemc.siemens.de). his is source with SunOS binaries.

NOTE: VHDL-NICE, VHDL-2-C, and VHDL-PROF currently support the following OS platforms: LINUX, SunOS, WIN32
